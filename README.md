# CLA-Assistant

Ez a projekt egy CLA-Assistant robotot tartalmaz, amely automatizálja a CLA (Contributor License Agreement) ellenőrzését és kezelését.

## Könyvtárstruktúra
- `src/` – Forráskód
- `tests/` – Tesztek
- `.github/` – GitHub workflow és Copilot utasítások
- `config.json` – Konfigurációs fájl

## Használat
A projekt célja, hogy egyszerűen integrálható legyen bármely GitHub repository-ba, és automatikusan ellenőrizze, hogy a hozzájárulók elfogadták-e a CLA-t.

## Fejlesztés
A forráskód a `src` mappában található, a tesztek pedig a `tests` mappában.

## Konfiguráció
A `config.json` fájlban adhatók meg a projekt beállításai.
